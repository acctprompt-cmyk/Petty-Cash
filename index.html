🔧 ตั้งค่าครั้งแรก
ทำครั้งเดียว ประมาณ 15 นาที — ไม่ต้องมีความรู้ด้านเทคนิค
1
สร้าง Google Cloud Project
ไปที่ console.cloud.google.com
→ คลิก Select a project (มุมบนซ้าย) → New Project
→ ตั้งชื่อ เช่น petty-cash → Create
2
เปิด Google Drive API
เมนูซ้าย → APIs & Services → Library
→ ค้นหา Google Drive API → คลิก → Enable
3
ตั้งค่า OAuth Consent Screen
APIs & Services → OAuth consent screen
→ เลือก External → Create
→ กรอกชื่อแอป เช่น Petty Cash App และ Support email
→ Save and Continue จนครบทุกหน้า
→ หน้า Test users → Add Users → ใส่ Gmail ของทุกคนที่จะใช้งาน → Save
4
สร้าง OAuth 2.0 Client ID
APIs & Services → Credentials
→ Create Credentials → OAuth client ID
→ Application type: Web application
→ Authorized JavaScript origins: ใส่ URL ที่จะ host เช่น https://yourapp.netlify.app
  (ถ้าทดสอบจากไฟล์โดยตรง เพิ่ม http://localhost ด้วย)
→ Create → คัดลอก Client ID
5
สร้าง Google Drive Folder
ใน Google Drive → + New → New Folder → ตั้งชื่อ PettyCash
→ เปิดโฟลเดอร์ → ดู URL:
drive.google.com/drive/folders/1AbCdEf...
→ คัดลอก ID ส่วนหลัง (ตัวอักษรยาวๆ)
6
Host ไฟล์นี้บน Netlify (ฟรี)
ไปที่ app.netlify.com/drop
→ ลากไฟล์ petty-cash-drive.html วางลงในหน้าเว็บ
→ ได้ URL เช่น https://random-name.netlify.app
→ นำ URL นี้ไปใส่ใน Authorized JS origins ของ Google Cloud (ขั้นตอน 4) ด้วย

🔥 Firebase Realtime Database (Real-time sync)

ค่าเหล่านี้เก็บเฉพาะใน browser ของคุณ (localStorage)
